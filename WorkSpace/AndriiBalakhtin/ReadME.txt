Io utilizzato ESP32, per questo progetto

	DHT11 Sensore  - Humidity + Temperature
	SH1106 	           - Display OLED
	SMD RGB          - LED RGB


Step 1
	Aprire Xampp creare DataBase (solo creare)
Step 2 
	Aprire Visual Studio C#, e dove string -> bisogno mettere vosto Database (Solo nome)
Step 3
	Aprire Visual Studio Code C++, e Upload the code (controlate pin se collegatto il vosto sensore)
	ma anche bissogno scaricare liberie
Step 4
	Quando carricato codice dentro, aprite C# e aprite Windows Form
Step 5
	pronto! (se volete potete entrate su sito locale se avete OLED display colegatto ) o senza con serial
	e potette entrate su sitto
